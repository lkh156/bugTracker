Bug Tracker Application [name under construction]

This application will have following features.

  1. Allow users to submit bug reports.
  2. Manager will review bug report and assign it to a developer.
  3. Developer will fix bugs and submit to manager for review.
  4. Manager will review the bugs then commit the changes.
  5. User had an option to be notified when the bug was fixed, if selected an update will be posted to the user.
